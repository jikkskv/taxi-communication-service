#taxi-communication-service
## Building a fat jar
```shell
cd ./taxi-communication-service
mvn clean package
```

## Testing the application
```shell
cd ./taxi-communication-service
mvn clean test
```

## How to Run
```shell
java -jar ./taxi-communication-service/taxi-communication-service.jar
```

## Running the application locally
```shell
mvn spring-boot:run
```
